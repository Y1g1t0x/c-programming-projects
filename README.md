# C Programming Projects

This repository contains basic C programming projects that I developed to improve my algorithm and problem-solving skills.

## Projects

### Prime Numbers Generator
Prints all prime numbers up to a given number.

### Fibonacci Sequence
Generates Fibonacci numbers.

### Factorial Calculation
Calculates the factorial of a number.

### Max-Min Finder
Finds the maximum and minimum values among given numbers.

### Even-Odd Checker
Determines whether a number is even or odd.

### Average Calculator
Calculates the average of given numbers.

### Basic Calculator
Performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
